## <samp>◆ &nbsp; About</samp>

I'm a **Full-Stack Developer** working at the intersection of **systems architecture** and **human-centered design**. My work spans **offline-first** desktop platforms, secure **SaaS** products, and production-grade **AI/RAG** pipelines.

Currently leading engineering at **Osys Software** — architecting localized payment ecosystems and AI-augmented automation for the Israeli market. In parallel, I founded **Resibilis**, a SaaS receipt platform built for Filipino freelancers, and contracted with **Totaledge.au** to build a dual-brand coaching platform with custom CRM tooling.

I build with **_Pagpalangga_** — the Hiligaynon word for *care*. Code that isn't just functional, but resilient, secure, and built to outlast the deploy.
